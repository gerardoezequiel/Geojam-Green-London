# Akif's GeoJam Enhancements (2025-2026)

This folder contains the complete advanced analytical suite for the "What Makes London Green?" challenge.

## Contents:
*   **Scripts/**: All Python scripts for data fetching (Deep Patch), normalization, land-cover classification, and strategic analysis.
*   **final_results/**: High-fidelity MSOA metrics and spatial data ready for DuckDB/DeckGL.
*   **strategic_maps/**: Definitive visualizations of Environmental Equity and Greening Hotspots.
*   **annual_verification_maps/**: Year-by-year land cover maps (2016-2025).
*   **london_greenness_10year_timelapse.gif**: Optimized animation for Discord/Presentations.
*   **PROJECT_LOG.md**: Complete technical audit of every refinement made.

## How to use:
The primary data for the front-end is in `final_results/london_final_msoa_metrics.csv`.
